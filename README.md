# 🚀 Chat2Order

Backend do sistema **Chat2Order**, uma API moderna desenvolvida para automatizar pedidos via chat, integrando processamento de mensagens, gerenciamento de pedidos e persistência de dados com alta performance.

## 📌 Sobre o projeto

O **Chat2Order Backend** é uma API construída para suportar um sistema de pedidos via chat, permitindo a criação, gerenciamento e processamento de pedidos de forma automatizada.

O projeto foi desenvolvido com foco em:

- Escalabilidade
- Organização de código
- Integração com banco de dados via Prisma
- Containerização com Docker

---

## ⚙️ Tecnologias utilizadas

- Node.js
- TypeScript
- Express
- Prisma ORM
- Docker
- PostgreSQL (ou outro banco configurado no `.env`)
- Jest
